# 앙상블 — Ensemble

여러 모델의 예측을 합쳐 단일 모델보다 높은 성능을 내는 기법.

## 앙상블의 핵심 원리

서로 다른 모델은 서로 다른 오류를 범함.
→ 합치면 오류가 상쇄되어 전체 정확도 향상.

## 모델 앙상블 (이 프로젝트 계획)

```
이미지 → yolov8n 예측 → 박스 세트 A
이미지 → yolov8s 예측 → 박스 세트 B
              ↓
           WBF 합산
              ↓
         최종 박스 세트
```

모델을 **합치는 게 아니라** 예측 결과(박스)를 합치는 것.
각 모델은 따로 학습되고 따로 저장됨.

## TTA와의 차이

| | 모델 앙상블 | [[concepts/TTA]] |
|---|---|---|
| 모델 수 | 여러 개 | 1개 |
| 다양성 원천 | 모델 구조/크기 차이 | 입력 이미지 변형 |
| 추론 횟수 | 모델 수 × 이미지 수 | 변형 수 × 이미지 수 |
| 학습 비용 | 모델 수만큼 학습 필요 | 추가 학습 불필요 |
