# 학습 안정화 기법

Ultralytics 래퍼가 기본 적용하는 기법들. 커스텀 학습 루프에서는 직접 추가해야 함.

## EMA (Exponential Moving Average)

매 스텝마다 모델 가중치의 이동 평균을 별도로 보관.

```python
ema_weight = 0.9999 × ema_weight + 0.0001 × 현재_weight
```

- 학습은 현재 모델로 하고, **평가는 EMA 모델**로 함
- 효과: val mAP가 에폭마다 출렁이지 않고 안정적으로 나옴
- decay 0.9999 = 최근 1만 스텝의 평균을 추적

## Gradient Clipping

역전파 중 gradient가 폭발적으로 커지면 loss가 NaN이 됨.

```python
# optimizer.step() 직전에 추가
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=10.0)
```

- gradient의 L2 norm이 max_norm을 초과하면 스케일 다운
- lr이 높을수록, 학습 초반일수록 효과적

## Mixed Precision (fp16)

float32 → float16으로 학습.

```python
from torch.cuda.amp import autocast, GradScaler

scaler = GradScaler()
with autocast():
    output = model(images)
    loss = criterion(output, targets)
scaler.scale(loss).backward()
scaler.step(optimizer)
scaler.update()
```

- GPU 메모리 절반 사용 → 배치 크기 2배 가능
- 속도 약 2배 (Tensor Core 활용)
- 정확도 손실 거의 없음
