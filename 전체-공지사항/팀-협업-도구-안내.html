김범진(PM) — Obsidian 기능

Git 자동화 (obsidian-git)

  • 자동 pull: 10분마다 자동으로 최신 내용 가져옴 (별도 조작 불필요)
  • 자동 commit + push: 10분마다 변경사항 자동 커밋 & 푸시 (별도 조작 불필요)
  • 수동 commit: 좌측 리본 또는 커맨드 팔레트에서 “Commit all changes” 실행 → 커밋 메시지 vault backup: 날짜시간 자동 입력
  • 수동 push: commit 후 “Push” 실행 (push 전 자동으로 pull 먼저 실행됨)
  • commit + push 시 Q&A 알림·자동 서명 등 pre-commit 훅이 같이 실행됨

파일 순서 변경 (file-shift)

Explorer 사이드바에서 파일/폴더를 드래그해서 원하는 순서로 배치 가능.

전체 검색 (Omnisearch)

vault 전체를 내용 기준으로 검색.

  • 단축키: Option + Space (Mac 기준, Windows 단축키 미확인)
  • 파일명 · 폴더명 · 본문 내용 전부 검색
  • Tab 키로 파일 검색 ↔ 볼트 전체 검색 전환
  • 퍼지 검색 지원 — 오타가 있어도 유사 결과 표시
  • 검색 결과에서 해당 위치 미리보기(excerpt) 제공

게시판

게시판/ 폴더. 자유게시판과 Q&A게시판 두 타입 지원.

파일 만드는 법: 게시판/ 폴더에 새 노트 생성 → Templater가 폴더를 보고 자동으로 템플릿 적용 → 타입(자유/Q&A) 선택 → 제목 입력 → 자동으로 게시판/[자유]-제목.md 또는 게시판/[Q&A]-제목.md 생성

자유게시판 ([자유]-제목.md)

  • 본문 섹션에 내용 작성 → commit + push 시 > [!note] 이름 callout 자동 서명
  • 댓글 섹션도 동일하게 자동 서명

Q&A게시판 ([Q&A]-제목.md)

  • 질문 섹션 작성 → commit + push → Discord에 “새 질문” 알림 (질문은 자동 서명 없음)
  • 답변 섹션 작성 → commit + push → > [!note] 이름 자동 서명
  • 댓글 섹션도 자동 서명
  • ⚠️ Discord에서 답변 → Obsidian 자동 반영은 미구현

Discord 알림 예시 (새 Q&A 파일 push 시):

❓ 새 질문이 등록됐어요! 아는 분 답변 부탁드립니다 🙏
제목: YOLOv8 설치 오류 해결 방법
─────────────────
pip install ultralytics 했는데
ERROR: Could not find a version... 뜨는데
어떻게 해야 하나요?
─────────────────

Q&A 칸반

Q&A 게시판 파일을 생성하면 자동으로 칸반 카드가 생성됨.

  • 칸반 보드에서 질문·답변 상태를 한눈에 확인 가능
  • 카드를 드래그해 상태(미답변 → 답변완료 등) 변경 가능
  • 카드 클릭 → 해당 Q&A 파일로 바로 이동해 질문·답변 내용 확인

공지사항

전체 공지사항/ 폴더. 팀 전체에 전달할 내용 작성.

파일 만드는 법: 전체 공지사항/ 폴더에 새 노트 생성 → Templater가 자동으로 템플릿 적용 → 파일명 자동으로 YYYY-MM-DD-공지.md 로 설정됨

  • 내용·댓글 섹션에 글 쓰고 commit + push 시 > [!note] 이름 callout 자동 서명
  • 대상 체크박스로 수신 대상 표시 가능

협업일지

협업일지/이름(역할)/ 폴더에 날짜별로 작성. 파일 만드는 법: 본인 협업일지/이름(역할)/ 폴더에 새 노트 생성 → Templater가 자동으로 템플릿 적용 홈 에서 팀원별 최근 일지 자동으로 모아볼 수 있음.

메모

메모/이름(역할)/ 폴더에 자유롭게 작성. 개인 작업 메모, 아이디어, 임시 정리 등 용도.

PR Discord 알림 (discord-pr-notify.yml)

PR 이벤트마다 Discord에 embed 메시지 자동 발송. 아웃풋 예시:

이벤트Discord 메시지색상
PR 생성🔔 새 PR이 열렸습니다!파랑
인라인 댓글💬 PR 코드에 인라인 댓글이 달렸습니다노랑
리뷰 approve✅ PR이 승인됐습니다!초록
변경 요청🔁 변경 요청이 달렸습니다빨강
PR 머지✅ PR이 머지됐습니다! + 머지한 사람 표시초록

embed 예시 (PR 생성 시):

🔔 새 PR이 열렸습니다!
#11 dataset.py 구현  (링크)
───────────────────────────────────
작성자       zipdid
브랜치       feature/11-dataset → main
내용         ████████  (클릭해서 펼치기)
  • 내용 필드: PR 본문에서 코드 블록 제거 후 200자 요약, 기본은 스포일러(클릭 전까지 가려짐)

협업일지 Discord 알림 (daily-reminder.yml)

cron-job.org를 통해 매 평일 19:00 KST 에 Discord에 자동 메시지 발송.

  • 아웃풋: 🌇 오늘도 고생 많으셨습니다! 협업일지 작성 잊지 마세요 📝
  • 알림 받으면 그날의 협업일지 작성하면 됨

GitHub 이슈 자동 동기화 (sync-issue-status.yml)

GitHub에서 이슈/PR 상태가 바뀌면 tasks/issue-{N}.md 의 st: 필드 자동 업데이트.

  • 이슈 closed → done / PR opened → in-review / PR merged → done
  • 아웃풋: 태스크 관리 의 태스크 현황 쿼리에 즉시 반영
